# Simple Energy Tools

[![License](https://img.shields.io/badge/license-Apache%202.0-blue.svg)](LICENSE)

This repository contains simple Python utilities for energy calculations.

## Functions

- `calculate_energy(power_kw, hours)`  
  Calculates energy consumption in kWh.

- `calculate_cost(energy_kwh, price_per_kwh)`  
  Calculates electricity cost.

## Testing

Tests are implemented using `pytest` and are automatically executed via GitHub Actions.
